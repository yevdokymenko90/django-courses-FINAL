# Django Courses Project

## Prerequisites

- Python 3.11
- pipenv (install using `pip install pipenv`)

## Create .venv folder

`mkdir .venv`

## Install dependencies

`pipenv install`

## Activate virtual environment

`pipenv shell`

## Launch server

`python manage.py runserver`

## Open Django web project in the web browser

`http://127.0.0.1:8000`
